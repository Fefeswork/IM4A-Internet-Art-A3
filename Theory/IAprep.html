REFLECTION

“Net Art’s spectrum is so wide that it is difficult to define…”(Ricci, 2020).
For the internet art, I decided to create a website game that mimics the idea of a whack-a-mole game:

For my idea, this concept would be used however instead of whacking a mole you are collecting treasure. Images of a treasure chest with gold in it would pop upon different grid plots of board and the player would click on it to score points, eventually, a pirate hat would appear and if the player clicks on it the game ended meaning they’d lost their treasure to pirates. Technically I used guidance from various websites to help the game function the way I wanted it to, I managed to make use of JavaScript to create this game. Technically the guidance from the different sources I used helped ease my concerns and created this was quite enjoyable. However when the site was deployed there were issues with the game functioning, I wasn’t quite sure what the issue was because all the script referencing was correct, I tried other methods and it still wouldn’t work once the site was deployed on GitHub which in turn “ruins” the experience I wanted to create. This idea helped me create the game experience I wanted as I view web games as a form of internet art, with the idea wanted to work with- I wanted to create a sense of exploration for the user so that once they reach the internet art section it almost feels like they are close to finding the treasure and then they are introduced to a game where they have to collect the treasure and they would feel accomplished.
